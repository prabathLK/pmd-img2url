API for Developers

Use this service programmatically. The API is designed to be simple, fast, and stable for use in your projects (like PRABATH-MD).

Upload Image (`POST /upload`)

Send a `multipart/form-data` request with a field named `file` (max 2MB).

// Returns: { url: "https://.../v/key" }

Create Paste (`POST /p/upload`)

Send a JSON request with `content` and optional `password` (MASTER_PASSWORD).

// Example: Normal Paste
fetch('/p/upload', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ content: 'Hello World' })
});

// Example: Encrypted Paste
fetch('/p/upload', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    content: 'My Secret Password', 
    password: 'prabath-md-master-password' 
  })
});
// Returns: { url: "https://.../p/key" }

Create Tracker (`POST /t/create`)

Send a JSON request with `trackKey` (unique) and optional `description` and `targetURL`.

fetch('/t/create', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    trackKey: 'sticker-cmd-usage', 
    description: 'Total .sticker uses'
  })
});
// Returns: { key, trackingURL, statsURL }

Track Click (`GET /t/:key`)

This is the URL to use. Accessing it increments the count and redirects (if `targetURL` was set) or returns a 1x1 pixel.

Get Tracker Stats (`GET /t/stats/:key`)

Returns a JSON object with the tracker's data, including the `count`.
