API for Developers

Use this service programmatically. The API is designed to be simple, fast, and stable for use in your projects (like PRABATH-MD).

Upload a File (`POST /upload`)

Send a `multipart/form-data` request with a field named `file` containing your image data (max 2MB).

// Example using fetch (JavaScript)
const fileInput = document.querySelector('input[type="file"]');
const formData = new FormData();
formData.append('file', fileInput.files[0]);

try {
  const response = await fetch('https://pmd-img2url.koyeb.app/upload', {
    method: 'POST',
    body: formData
  });
  
  const data = await response.json();
  
  if (!response.ok) {
    throw new Error(data.error);
  }

  console.log('Success:', data.url);
  
} catch (error) {
  console.error('Error:', error.message);
}

Success Response (Download/View URL)

A successful upload will return a `200 OK` status and a JSON object containing the "smart" URL. This is the URL you must use, as it tracks accesses for the auto-delete script.

{
  "url": "https://pmd-img2url.koyeb.app/v/a1b2c3d4e5f6.jpg"
}

View / Download (`GET /v/:key`)

When you access the `url` provided, the server will log the access (resetting the 30-day delete timer) and then instantly redirect you to the high-speed, cached Cloudflare R2 file.

Error Response (e.g., 10GB Limit Reached)

If the upload is blocked (e.g., storage full), you will get a non-200 status and a JSON error.

{
  "error": "Storage limit exceeded (10GB). Cannot upload new files..."
}
