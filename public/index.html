API for Developers

Upload Endpoint

Send a `multipart/form-data` request to the endpoint below with a field named `file` (max 2MB).

POST ...

JavaScript Example

const formData = new FormData();
formData.append('file', yourFileObject);

const response = await fetch(..., {
  method: 'POST',
  body: formData
});
const data = await response.json();

View / Download Endpoint

Access the `url` from the success response. This logs the access (for the 30-day auto-delete) and redirects to the high-speed R2 file.

GET /v/:key
