API for Developers

Upload Endpoint

Send a `multipart/form-data` request to the endpoint below with a field named `file` (max 2MB).

POST

JavaScript Example

// Example: JavaScript Fetch
const formData = new FormData();
formData.append('file', yourFileObject);

const response = await fetch(, {
  method: 'POST',
  body: formData
});
const data = await response.json();

// Success: { url: "https://.../v/key" }
// Error:   { error: "File too large..." }

View / Download Endpoint
