API for Developers

Upload a File (`POST /upload`)

Send a `multipart/form-data` request with a field named `file` (max 2MB).

// Example: JavaScript Fetch
const formData = new FormData();
formData.append('file', yourFileObject);

const response = await fetch('/upload', {
  method: 'POST',
  body: formData
});
const data = await response.json();

// Success: { url: "https://.../v/key" }
// Error:   { error: "File too large..." }

View / Download (`GET /v/:key`)

Access the `url` from the success response. This logs the access (for the 30-day auto-delete) and redirects to the high-speed R2 file.
